# Rent-Car-Sales-Data-Analysis
Car Rental Sales Data Analysis using Excel and Power BI

##  Project Overview
This project analyzes car rental booking data to identify sales trends, profit patterns, and customer behavior.

##  Tools Used
- Microsoft Excel (Data Cleaning)
- Power BI (Interactive Dashboard & Data Visualization)

##  Key Insights
- Monthly Sales Trend Analysis
- Profit vs Cost Analysis
- Booking Channel Performance
- Weekend vs Weekday Booking Analysis

##  Files Included
- Cleaned Dataset (CSV)
- Power BI Report File
- Dashboard Screenshots
